# Stock-Query-System
In this project, I have created e a webpage that allows you to search for stock information using the Tiingo Stock API, and the results will be displayed in both tabular format and charts format using HighCharts. You will also provide news articles for the selected stock using the News API.
The full description of the project in included in the repository along with the code for frontend- written using HTML5, CSS3, JavaScript, DOM, JSON format and
XMLHttpRequest object and the backend written in- Python/Flask. 
The project was then deployed on AWS ElasticBeanStalk.
